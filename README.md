# Простая система слежения за взглядом

Учебная программа на Python, которая использует веб-камеру, находит лицо и глаза, выделяет предполагаемый центр зрачка и оценивает направление взгляда.

## Что улучшено

- добавлена калибровка центрального положения взгляда;
- добавлено сглаживание результата по нескольким кадрам;
- добавлена ручная настройка порога выделения зрачка;
- добавлена запись измерений в CSV-файл;
- добавлено экспериментальное управление курсором взглядом;
- интерфейс теперь показывает количество найденных глаз, центр калибровки и текущие параметры.

## Установка

Рекомендуемый вариант:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

Если виртуальное окружение не нужно:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python eye_tracking.py
```

Если камера не открывается:

```bash
python eye_tracking.py --camera 1
```

Запуск с записью результатов:

```bash
python eye_tracking.py --log gaze_log.csv
```

Запуск с управлением курсором:

```bash
python eye_tracking.py --control-cursor
```

## Управление

- `C` - калибровать центр взгляда. Нужно смотреть прямо в камеру или в центр экрана.
- `M` - включить или выключить управление курсором, если программа запущена с `--control-cursor`.
- `+` и `-` - изменить порог выделения зрачка, если программа плохо видит темную область глаза.
- `Q` - выйти из программы.

## Как работает алгоритм

1. Кадр с камеры зеркально отражается и переводится в оттенки серого.
2. OpenCV улучшает контраст через выравнивание гистограммы.
3. Лицо определяется каскадом `haarcascade_frontalface_default.xml`.
4. В верхней половине лица ищутся глаза каскадом `haarcascade_eye.xml`.
5. В области глаза применяется размытие, пороговая бинаризация и морфологическая фильтрация.
6. Самая подходящая темная область считается кандидатом на зрачок.
7. Положение зрачка переводится в относительные координаты от `0` до `1`.
8. После калибровки программа сравнивает текущее положение с центральным и выводит направление взгляда.
9. Для устойчивости итоговое направление сглаживается по истории последних кадров.
10. Если включено управление курсором, направление взгляда превращается в небольшой шаг курсора по экрану.

## Ограничения

Это демонстрационная система, а не медицинский или промышленный eye tracker. На результат влияют освещение, очки, поворот головы, качество камеры и расстояние до экрана. Для более точной версии можно использовать facial landmarks, инфракрасную подсветку или нейросетевую модель.
