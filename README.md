# Простая система слежения за взглядом

Учебная программа на Python, которая использует веб-камеру, находит лицо и глаза, а затем грубо оценивает направление взгляда по положению темной области зрачка внутри области глаза.

## Возможности

- захват видео с веб-камеры;
- поиск лица и глаз через Haar Cascade из OpenCV;
- выделение предполагаемого центра зрачка;
- отображение направления взгляда: `left`, `right`, `up`, `down`, `center`;
- визуальная разметка лица, глаз и зрачков в окне программы.

## Установка

Лучше запускать проект в отдельном виртуальном окружении:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

Если виртуальное окружение не нужно, зависимости можно поставить напрямую:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python eye_tracking.py
```

Если камера не открывается, можно попробовать другой номер:

```bash
python eye_tracking.py --camera 1
```

Для выхода нажмите `Q`.

## Как работает алгоритм

1. Кадр с камеры переводится в оттенки серого.
2. OpenCV находит лицо с помощью `haarcascade_frontalface_default.xml`.
3. В верхней половине лица ищутся глаза с помощью `haarcascade_eye.xml`.
4. Для каждого глаза применяется размытие и бинаризация методом Оцу.
5. Самая крупная темная область считается кандидатом на зрачок.
6. Положение центра этой области сравнивается с шириной и высотой глаза.
7. По относительному положению выводится примерное направление взгляда.

## Ограничения

Это демонстрационная система, а не медицинский или промышленный eye tracker. На качество сильно влияют освещение, положение головы, очки, качество камеры и расстояние до экрана. Для более точной системы можно добавить калибровку, использовать landmarks лица или нейросетевую модель.
