# Простая система слежения за взглядом

Учебная программа на Python, которая использует веб-камеру, находит лицо и глаза, выделяет предполагаемый центр зрачка и оценивает направление взгляда.

## Что улучшено

- добавлена калибровка центрального положения взгляда;
- добавлено сглаживание результата по нескольким кадрам;
- поиск глаз и радужки переведен на MediaPipe Face Landmarker;
- добавлена запись измерений в CSV-файл;
- добавлено управление курсором через калибровку точек экрана;
- интерфейс оформлен как окно Gaze Studio с HUD, статусами, confidence и мини-картой взгляда.

## Установка

Для версии с MediaPipe рекомендуется Python 3.10-3.12. На Python 3.13 пакет `mediapipe` может быть недоступен через pip.

Рекомендуемый вариант:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

Если виртуальное окружение не нужно:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python eye_tracking.py
```

Если камера не открывается:

```bash
python eye_tracking.py --camera 1
```

Запуск с записью результатов:

```bash
python eye_tracking.py --log gaze_log.csv
```

Запуск с уже включенным управлением курсором:

```bash
python eye_tracking.py --control-cursor
```

Чтобы собрать модель прямого попадания, нажми `C` и пройди калибровку по точкам. После этого курсор начнет следовать за точкой взгляда.

Если курсор дрожит, попробуй чуть увеличить `--cursor-smoothing` или заново пройти калибровку.

## Управление

- `C` - запустить калибровку по точкам экрана.
- `M` - включить или выключить управление курсором.
- `R` - сбросить калибровку и модель курсора.
- `Q` - выйти из программы.

Во время калибровки смотри на целевые точки и старайся держать голову максимально ровно. После завершения курсор начинает двигаться к месту, куда ты смотришь.

## Как работает алгоритм

1. Кадр с камеры зеркально отражается и передается в MediaPipe Face Landmarker.
2. MediaPipe находит landmarks лица, контуры глаз и landmarks радужки.
3. Для каждого глаза вычисляется центр радужки.
4. Центр радужки переводится в относительные координаты внутри контура глаза от `0` до `1`.
5. Показания двух глаз усредняются с учетом confidence.
6. После калибровки программа сравнивает текущее положение с центральным и выводит направление взгляда.
7. Для устойчивости итоговое направление сглаживается по истории последних кадров.
8. Если включено управление курсором, направление взгляда превращается в небольшой шаг курсора по экрану.

## Ограничения

Это демонстрационная система, а не медицинский или промышленный eye tracker. На результат влияют освещение, очки, поворот головы, качество камеры и расстояние до экрана. Для более точной версии можно добавить многоточечную калибровку экрана, инфракрасную подсветку или специализированную нейросетевую модель.

При первом запуске приложение использует модель `face_landmarker.task` из `.cache/mediapipe`. Если файла нет, программа попробует скачать его автоматически.
